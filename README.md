# Code21 UI

Vite + React + Typescript



## Quickstart

1. Clone this repository.
   ```
   git clone git@gitlab.emacs.energy:code21/frontend.git
   ```
2. Install required dependencies
   ```
   npm install
   ```
3. Start project
   ```
   npm run dev
   ```

## Deploy

1. Add file `service-account.json` to project directory

2. Deploy app
   ```
   npm run deploy
   ```
